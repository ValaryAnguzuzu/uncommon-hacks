# Sink Or Swim Prototype

This Godot project contains a first playable prototype for **Sink Or Swim**, a 2D dashboard strategy game about surviving recruiting season before graduation.

## How to Test

1. Open Godot.
2. Click **Import**.
3. Select `project.godot` in this folder.
4. Open `testing.tscn`.
5. Press the play button.

The project main scene is already set to `res://testing.tscn`, so pressing play should launch the prototype directly.

## Current Prototype Loop

- Spend 3 action points each week.
- Improve resume, build projects, practice interviewing, apply to jobs, work, or rest.
- End the week to pay expenses and resolve applications.
- Win by getting an offer.
- Lose from burnout, debt crisis, or reaching graduation without an offer.
